要件定義 (Requirements)
=======================

1. 概要
-------
現実空間の手書き数字（紙に書かれた文字など）を写真として取り込み、AIによって数字(0-9)を判定するアプリケーション。

2. 機能要件
-----------
*   **画像アップロード**: ユーザーは手元の端末から画像ファイル(JPG, PNG等)をアップロードできる。
    *   Webブラウザ経由での利用に対応する。
*   **画像前処理**: アップロードされた画像に対して、以下の自動処理を行う。
    *   グレースケール変換
    *   ノイズ除去
    *   2値化（文字成分の抽出）
    *   文字領域の切り出し(Crop)
    *   MNIST形式(28x28ピクセル, 黒背景, 白文字)への変換・リサイズ
*   **AI判定**: 前処理された画像を入力とし、CNNモデルを用いて数字を予測する。
    *   予測された数字を表示する。
    *   その予測の確信度(%)を表示する。
*   **結果可視化**: 元画像と、AI入力用に加工された画像を並べて表示する。

3. 非機能要件
-------------
*   **プラットフォーム**: Fletを使用したクロスプラットフォーム(Web, Desktop)対応。
*   **パフォーマンス**: 画像アップロードから判定まで数秒以内で完了すること。
*   **精度**: 丁寧に書かれた数字であれば高い確率で正解すること。
